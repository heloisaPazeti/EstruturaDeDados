# AVL - Default

Regular AVL Tree, uses strings as generic Type Data. 

### Contains

- Insert
- Remove
- RR, LL, RL, LR Rotate
- Search
- Print in Order
- Print Pos Order
- Print Pre Order
- Total Nodes
